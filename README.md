# Etnya | Boutique Remote Work Stays in Mérida

Premium, design-led apartments and community for remote workers in Mérida, Yucatán. Experience boutique hospitality with professional standards.

## Project Overview
This project is a modern, high-performance web application built for **Etnya**, a remote work community in Mérida. It features a curated property catalog, community-focused lifestyle sections, and a premium booking experience inspired by industry leaders like Outsite, Locke, and Marriott Homes & Villas.

## Tech Stack
- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4
- **Animations**: Motion (framer-motion)
- **Routing**: React Router Dom
- **Icons**: Lucide React
- **AI Integration**: Google Gemini API (via @google/genai)

## Key Features
- **Editorial Hero**: Calm, lifestyle-focused entry point.
- **Smart Booking Bar**: Compact and utility-driven search interface.
- **Property Catalog**: 15+ curated residences with rich metadata.
- **Detailed Property Pages**: Professional layout with bedroom configurations, amenity blocks, and house rules.
- **Community Layer**: Sections dedicated to the remote work lifestyle and neighborhood belonging.

## Getting Started
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Add your `GEMINI_API_KEY` to a `.env` file.
4. Run `npm run dev` to start the development server.
